### 02 ジョブとは
- コマンドを一時停止する
  ```
  (Ctrl + z)
  $
  ```

- 現在のジョブの一覧を表示する
  ```
  $ jobs
  ```

  - ジョブ一覧にプロセスIDも表示
    ```
    $ jobs -l
    ```

- ジョブをフォアグラウンドにする
  ```
  $ fg %<ジョブ番号>
  ```

- ジョブをバックグラウンドにする
  ```
  bg %<ジョブ番号>
  ```

  - 最初からバックグラウンドで行う場合
    ```
    $ <コマンド> &
    ```

- ジョブの状態遷移<br>
  | 状態 | 内容 |
  | :---: | :--- |
  | フォアグラウンド | ユーザが対話的に操作しながら処理が実行されている状態 |
  | バックグラウンド | ユーザが対話的に操作せずに処理が実行されている状態 |
  | 停止 | 処理を一時的に中断している状態 |
