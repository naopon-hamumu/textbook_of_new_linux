### 04 繰り返しを指定するメタ文字
- 繰り返し
  ```
  '<文字列><繰り返ししたい文字列>*<文字列>'
  ```

- 拡張正規表現
  ```
  -E '<文字列><繰り返ししたい文字列>+<文字列>'
  ```

- 繰り返し回数の指定をするメタ文字
  | 基本正規表現 | 拡張正規表現 | 意味 |
  | :---: | :---: | :--- |
  | * | * | 0回以上の繰り返し |
  | なし | + | 1回以上の繰り返し |
  | なし | ? | 0回または1回の繰り返し |
  | \\{m,n\\} | {m,n} | m回以上n回以下の繰り返し |
  | \\{m\\} | {m} | ちょうどm回の繰り返し |
  | \\{m,\\} | {m,} | m回以上の繰り返し |
