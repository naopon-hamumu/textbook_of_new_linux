### 03 シェル変数
- 変数の設定
  ```
  $ <変数名>='<値>'
  ```
  [![Image from Gyazo](https://i.gyazo.com/e152526d16de5dc7db1e37ec65440557.png)](https://gyazo.com/e152526d16de5dc7db1e37ec65440557)<br>

  ※ =の左右にスペースをつけてはいけない<br>
  [![Image from Gyazo](https://i.gyazo.com/fc9d09daaa5c7282fa5491717d095c3b.png)](https://gyazo.com/fc9d09daaa5c7282fa5491717d095c3b)

  - 変数を参照
    ```
    $ <コマンド> $<変数>
    ```

- プロンプト設定
  ```
  $ PS1='<プロンプト>'
  ```
  [![Image from Gyazo](https://i.gyazo.com/4089deae12fa91210a1689593b99902e.png)](https://gyazo.com/4089deae12fa91210a1689593b99902e)

  - プロンプトで利用できる記号<br>
    | 記号 | 内容 |
    | :---: | :--- |
    | \d | 「曜日 月 日」という形式の日付 |
    | \h | ホスト名のうち、最初の.までの部分 |
    | \H | ホスト名 |
    | \n | 改行 |
    | \t | HH:MM:SS形式の現在時刻 |
    | \u | ユーザ名 |
    | \w | カレントディレクトリ |
    | \W | カレントディレクトリの末尾のディレクトリ名 |
    | \$ | rootユーザの場合は#、それ以外のユーザの場合は$ |
    | \\ | \ |

- LANG（ロケール）
  - 現在のロケールを確認
    ```
    $ echo $LANG
    ```

  - システムがサポートしているロケールを確認
    ```
    $ locale -a
    ```

  - アメリカ英語に設定する
    ```
    $ LANG=en_US.UTF-8
    ```

- その他のシェル変数
  - コマンドライン履歴関連のシェル変数
    | シェル変数名 | 内容 |
    | :---: | :--- |
    | HISTFILE | コマンドライン履歴を保存するファイル名。デフォルト値は~/.bash_history |
    | HISTFILESIZE | 履歴ファイルに保存するコマンドライン履歴の最大行数 |
    | HISTSIZE | コマンドライン履歴を保持する最大行数 |

  - シェル状態関連のシェル変数
    | シェル変数名 | 内容 |
    | :---: | :--- |
    | HOME | ホームディレクトリ |
    | SHELL | ログインシェルのパス |
    | PWD | カレントディレクトリ |
