### 02 トラブル時には
- キーボードからの操作を一切受け付けない
  | コマンド | 内容 |
  | :---: | :--- |
  | Ctrl + s | 画面表示をロックする |
  | Ctrl + q | 画面表示のロックを解除する |

- コマンドが終了せず、プロンプトが返ってこない
  | コマンド | 内容 |
  | :---: | :--- |
  | Ctrl + c | 実行中のコマンドを強制終了 |

- プロンプトが文字化けしている
  | コマンド | 内容 |
  | :---: | :--- |
  | Ctrl + l | 画面を削除する |
